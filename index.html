DiteCross

Discord-Telegram Cross messaging system

Download as .zip Download as .tar.gz View on GitHub

What is it?

DiteCross is a twin bot that helps Telegram and Discord users connect to each other. Currently, it uses a handshake system to connect one Telegram private chat/group with a Discord channel. DiteCross must be added to each Telegram and Discord groups in order to process your messages and relay them to the linked chat.

How do I make the handshake?

  1. To get started, add DiteCross to your Discord server
  2. Contact @DiteCross_bot on Telegram and, if you have a group, add the bot to it
  3. Type the command /handshake on Telegram chat
  4. The bot will ask What is the server and the channel you want to handshake?. Simply send a message like @MyFunnyServer#crosschannel where MyFunnyServer is the target Discord's server name and crosschannel the target channel name
  5. Go to Discord and type /handshake to confirm and link the groups
  6. Enjoy!

Note: If you want to revert back, you can type anytime the command /cancel. DiteCross will confirm with a message like Handshake cancelled.

How do I self host?

DiteCross is Open Source, so you can self host it by yourself. Simply go to the technical page and follow the guide.
